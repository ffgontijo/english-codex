# Contributing

Contributions are welcome when they make the Codex clearer, more accurate or easier to study.

## Good contributions

- clearer explanations;
- better learner examples;
- corrections to grammar or wording;
- new A0 to C2 pages;
- better Cambridge preparation notes;
- accessibility improvements;
- layout improvements that preserve readability.

## Content rules

- Write content in English.
- Keep explanations simple and useful.
- Avoid copyrighted material.
- Do not add private information.
- Prefer official references for exam details.

## Page format

Add pages in `data/pages.json` using the existing schema.
