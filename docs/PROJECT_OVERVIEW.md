# Project Overview

## What English Codex is

English Codex is a structured English-learning website for Brazilian learners.

It guides learners from **A0 to C2** while helping them build a personal notebook with explanations, examples, practice, error correction and revision.

The project combines:

- CEFR progression;
- Cambridge exam awareness;
- active recall;
- spaced review;
- original examples;
- Portuguese-English comparison;
- pop-culture learning hooks;
- teacher-friendly structure;
- PHP partials for maintainable layout;
- Docker-based self-hosting.

## Core principle

The project is built around this learning cycle:

```text
Understand → Connect → Write → Produce → Correct → Review → Use
```

The learner should not simply read the website. The learner should transform the content into a personal notebook.

## What the project is not

English Codex is not:

- a grammar dump;
- a PDF download collection;
- a replacement for official Cambridge exam documentation;
- a copied textbook;
- a passive reading website;
- a magical fluency button, because unfortunately physics still exists.

## Target audience

The main audience is Brazilian learners of English who want:

- structured progression;
- explanations in Portuguese;
- examples connected to real life;
- guidance for building a notebook;
- preparation for Cambridge exams;
- help with common Portuguese-to-English mistakes.

Secondary audiences:

- teachers;
- tutors;
- academic project reviewers;
- self-taught learners;
- learners interested in pop culture and RPG.

## Educational pillars

### 1. Active learning

Learners must produce language, not just consume explanations.

### 2. Notebook-based study

The notebook is the learner's personal knowledge system.

### 3. Brazilian learner focus

The project explicitly addresses interference from Portuguese.

### 4. CEFR progression

Levels are organized from A0 to C2.

### 5. Cambridge awareness

Cambridge sections help learners understand exam expectations from A2 onward.

### 6. Culture as context

Films, books, series, music, games and RPG are used as learning bridges.

## Definition of success

The project succeeds if a learner can:

1. identify their level;
2. study the appropriate page;
3. write a notebook entry;
4. create original examples;
5. practise the skill;
6. record mistakes;
7. review later;
8. progress to the next level with more autonomy.
