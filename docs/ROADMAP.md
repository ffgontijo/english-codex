## Not recommended right now

Avoid adding too early:

- user login;
- database;
- CMS;
- complex build system;
- heavy JavaScript framework;
- analytics;
- AI-generated exercises without review.

The current project benefits from being simple. Do not bury a clean educational site under machinery just because modern development has commitment issues.

## Guiding principle

Every future feature should answer:

```text
Does this help the learner study better, practise more, review more consistently or understand more clearly?
```

If not, it is probably decoration wearing a productivity costume.
