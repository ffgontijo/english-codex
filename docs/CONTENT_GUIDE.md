# Content Guide

## Purpose

This guide defines how content should be written and maintained in English Codex.

The goal is consistency. Without a content guide, every page slowly becomes its own tiny kingdom with its own laws. Humanity has tried this. Results vary.

## Voice and tone

The content should be:

- clear;
- direct;
- educational;
- practical;
- lightly informal;
- friendly to Brazilian learners;
- structured enough for teachers.

Humor is allowed, but clarity wins.

## Language

The website is primarily written in Portuguese for Brazilian learners.

English examples should be:

- correct;
- natural;
- level-appropriate;
- original;
- connected to real use.

## What every level page should include

Each level page should include:

1. level hero;
2. purpose of the level;
3. practical goals;
4. core grammar/content topics;
5. vocabulary;
6. real-life and pop-culture examples;
7. skill development;
8. writing guidance;
9. speaking guidance;
10. notebook guidance;
11. missions/practice tasks;
12. common mistakes;
13. checklist to move forward.

From A2 onward, include Cambridge exam relevance where appropriate.

## Brazilian learner focus

Always consider common Portuguese interference.

Examples:

| Portuguese tendency | Better English |
| --- | --- |
| `I have 26 years` | `I am 26 years old` |
| `I am agree` | `I agree` |
| `I have a doubt` | `I have a question` |
| `make a research` | `do research` |
| `depends of` | `depends on` |
| `in the other hand` | `on the other hand` |

## Original examples only

Use original examples.

Allowed:

```text
The detective was reading the report when the lights went out.
```

Avoid copying:

- song lyrics;
- book passages;
- film scripts;
- subtitles;
- textbook exercises;
- long copyrighted excerpts;
- protected images.

Pop culture can inspire examples, but examples should be original.

## Pop culture usage

Good usage:

```text
The spaceship is old, but it still works.
```

Better at higher levels:

```text
The final scene works because it relies on silence rather than exposition.
```

Avoid:

- copying actual dialogue;
- reproducing lyrics;
- using copyrighted plot summaries too closely;
- making the site dependent on one franchise.

## Writing examples

Writing examples should be short enough to read but rich enough to model structure.

For B2-C2, writing examples should show:

- argument;
- contrast;
- register;
- cohesion;
- style;
- precision.

## Cambridge content

Cambridge sections should explain:

- what the exam evaluates;
- what skills are required;
- how to prepare;
- how to use the notebook;
- how to diagnose mistakes.

Do not invent exam rules. Refer learners to official Cambridge sources for updated details.

## Content review checklist

Before publishing content:

- [ ] Is the explanation clear?
- [ ] Is the level appropriate?
- [ ] Are examples original?
- [ ] Are Brazilian learner traps included?
- [ ] Is there practical output?
- [ ] Is there notebook guidance?
- [ ] Are there common mistakes?
- [ ] Are there no copyrighted excerpts?
- [ ] Are Cambridge references accurate and official?
- [ ] Does the section help the learner actually use English?
