# Content Style Guide

## Language

Write the website content in English.

Use plain, clear explanations. The learner should understand the idea quickly, then use the examples to practise.

## Recommended page anatomy

Each lesson should usually contain:

1. Main idea
2. Structure or pattern
3. Natural examples
4. Brazilian traps
5. Memory hook
6. Copy by hand
7. Practice
8. Cambridge connection, when relevant

## Card classes

Use these classes consistently:

- `rule-card`: main rule or principle;
- `info-card`: explanation or context;
- `example-card`: natural examples;
- `trap-card`: common mistakes;
- `pt-card`: Portuguese-to-English interference;
- `upgrade-card`: more advanced alternatives;
- `nerd-card`: memory hook;
- `copy-card`: notebook-ready content;
- `practice-card`: tasks and drills;
- `exam-card`: Cambridge or exam-related notes;
- `warning-card`: safety, scope or important warnings.

## Copyright hygiene

Allowed:

- general references to RPG, sci-fi, fantasy and pop culture;
- generic terms such as quest, skill tree, boss fight, map, guild, inventory;
- original analogies.

Avoid:

- franchise logos;
- official images;
- copied character art;
- long quotes;
- lyrics;
- copyrighted text excerpts.

Because apparently making an educational site should not require summoning a legal demon.
