# Cambridge and References

## Purpose

English Codex includes Cambridge exam awareness as part of the learning path.

The goal is to help learners understand how their level connects to internationally recognized exams, without turning the entire project into exam worship. Exams are useful benchmarks, not sacred relics.

## How Cambridge should appear in the Codex

Each relevant level page should include a Cambridge section explaining:

- what the exam evaluates;
- which skills matter;
- what tasks the learner should practise;
- what to add to the notebook;
- how to diagnose errors;
- where to find official information.

## Important rule

The Codex should not claim to replace official Cambridge documentation.

Exam format, scoring, dates, fees and policies may change.

Always direct learners to official Cambridge sources.

## Recommended official references

Use official and reliable sources such as:

- Cambridge English;
- CEFR / Council of Europe;
- British Council LearnEnglish;
- English Profile;
- Cambridge Dictionary.

## Cambridge preparation philosophy

The project should treat exam preparation as a cycle:

```text
Learn the language → Understand the task → Practise deliberately → Diagnose mistakes → Review → Simulate
```

Doing random exam exercises without analysis is not preparation. It is just panic with a timer.

## References page standards

The `resources.php` page should remain transparent about references.

It should include:

- what each source is useful for;
- why it matters;
- how learners can use it;
- editorial rule about copyright.

## Copyright reminder

Do not reproduce:

- official exam papers without permission;
- copyrighted textbook tasks;
- song lyrics;
- long book excerpts;
- film or series scripts;
- protected images.

Use links to official sources and create original examples.
