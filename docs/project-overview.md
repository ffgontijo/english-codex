# Project Overview

**The English Codex** is a public English learning website focused on helping learners build a handwritten study system.

The project is not designed as a printable book first. It is designed as a readable website that guides the learner through explanations, examples, practice prompts and notebook-ready sections.

## Mission

Help learners study English from A0 to C2 through:

- structured grammar;
- listening, speaking, reading and writing practice;
- vocabulary and pronunciation systems;
- Cambridge preparation;
- active recall;
- handwritten notes;
- personal examples;
- review cycles.

## Identity

The tone is educational, direct and lightly nerdy. RPG, fantasy, sci-fi and pop-culture ideas can be used as learning metaphors.

Do not depend on copyrighted characters, logos, franchise images or long quotes. The vibe is allowed. The lawsuit bait is not.
